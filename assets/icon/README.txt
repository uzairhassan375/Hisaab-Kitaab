1. Place your app icon here as: app_icon.png
   - Recommended size: 1024x1024 pixels
   - Format: PNG
   - Background: Transparent or solid color
   - Design: Simple, recognizable icon

2. After placing your icon, run these commands:
   
   flutter pub get
   flutter pub run flutter_launcher_icons

3. That's it! The package will automatically:
   - Generate all required sizes for Android
   - Generate all required sizes for iOS
   - Generate all required sizes for Web
   - Place them in the correct locations

4. Clean and rebuild:
   flutter clean
   flutter run

For detailed instructions, see: APP_ICON_SETUP.md in the root directory
